# Design Mockup

Interactive mockup for the CPMAI Prep platform. Built with Vite + React + Tailwind.

## Run
```bash
npm install
npm run dev      # http://localhost:5173
```

## What's inside
Toggle **Learner ↔ Admin** in the sidebar. Demonstrates:
- Learner dashboard, AI tutor with daily quota counter, practice quiz, mock exams, progress
- Admin dashboard with cost tracking, LLM provider management, runtime settings,
  subscription tiers, security (2FA + key rotation), users, audit log
